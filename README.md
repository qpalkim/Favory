## 📌 프로젝트 소개

<img width="1080" alt="mockup-2" src="https://github.com/user-attachments/assets/07d07914-e8ad-4237-8a42-ac9aa06f9f29" />

<ul>
  <br/>
  <li>Favorite(좋아하는 것) + -ory(장소, 저장소)의 합성어로, 좋아하는 것들을 모아두는 공간이라는 의미</li>
  <li>우리는 누군가를 만날 때 흔히 서로의 취향을 묻지만, 정작 내 취향을 정리해 본 적은 많지 않으며 쉽게 떠오르지 않음</li>
  <li>그래서 좋아하는 콘텐츠를 한눈에 모아보고, 나만의 취향을 기록할 수 있는 공간을 만들고자 프로젝트를 기획</li>
</ul>

<br/>

### 📄 페이지 소개

- **랜딩 페이지** : 서비스 소개 및 사용자 흐름 안내
- **감상평 목록 페이지** : 등록된 모든 감상평을 목록형으로 조회
- **감상평 상세 페이지** : 감상평 상세 조회 및 작성자 프로필 조회, 댓글 작성/수정/삭제 기능
- **감상평 등록 페이지** : 작품 제목 검색 및 선택 시, Open API를 통해 작품 정보 조회 후 감상평 작성 가능
- **검색 페이지** : 키워드/태그/닉네임 검색 기능, 최근 검색어 제공
- **프로필 페이지** : 프로필 수정 기능 및 내가 작성한 감상평/댓글 목록 조회, 다른 사용자 프로필 조회

<br/>

### 🔑 주요 기능

- 음악, 영화, 드라마, 도서 중 하나의 카테고리를 선택하여 Open API를 통해 작품 정보를 불러와 감상평 작성 가능
- 다른 사용자의 감상평 조회 및 댓글 작성으로 의견을 나누며 소통할 수 있는 커뮤니티 기능 제공
- 반응형 디자인으로 모바일, 태블릿, PC 환경 지원
- 사용자 친화적인 로딩/에러 처리
- 실시간 토스트 알림 서비스
- Google OAuth 기반 간편 로그인 기능 구현
